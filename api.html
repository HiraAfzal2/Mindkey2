Vacancy

Vacancies are created in the MindKey by HR users, and can be retrieved from the API in order to be displayed on a webpage.

Get All Vacancies - Detailed

Retrieves all the vacancies in detail.

path Parameters
customerId
required
string <uuid>

A unique ID that identifies the customer.

query Parameters

string

string

Indicates the region name.

string

Determines the vacancy area name.

boolean
Deprecated

Indicates if external access is allowed or not.

boolean
Deprecated

Indicates if internal access is allowed or not.

string (EnumAccess)
Enum: "External" "Internal" "All"

Specifies the type of access.

IncludeAdvertisementText
boolean

Indicates whether advertisement text has been added or not.

PerDate
string <date-time>

Specifies the date and time.

SortDescending
boolean

Sorts the vacancy list in descending order.

Unsolicited
string (EnumVacancyListUnsolicited)
Enum: "Bottom" "Top" "Hide" "Only"

Displays desired vacancies through customizable options.

SortField
string (EnumVacancyListSortField)
Enum: "Description" "StartDate" "EndDate" "Priority"
